Enter the number of questions in the text box above on the right. This will create a table showing percentages for every score. If you give partial credit for effort, you may want to check the "0.5 step increments" checkbox. You can type the number of correct answers for a particular student in the left text box, and that score will be shown (useful for tests with so many questions the chart will not fit on the page)
